## 開發環境

- Windows Home 10 64bit
- C# `6.0.300`
- Unity `2021.3.1f1`
- Visual Studio Code `2017` Community

---

## 如何執行

> 預設 ***Windows Home 10 64bit*** 作業系統  

步驟 一：執行。  

- 方法一：下載任意版本，最好是 **Atmosphere Guardian v7-Final-Pro**  
>
- 方法二：點兩下 `AtmosphereGuardian.exe`
